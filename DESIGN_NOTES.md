# Design Notes - Geigenbau-Meisterin Website Redesign

## Extracted Color Scheme

### Primary Colors
- **Deep Red/Maroon (Primary Background)**: `#720505` / `rgba(114,5,5,1)`
  - Used for body background
  
- **Dark Blue/Charcoal (Header/Footer)**: `#27313E` / `rgba(39,49,62,1)`
  - Used for header, footer, navigation background
  - Also used for mobile menu background

### Neutral Colors
- **Gray (Secondary)**: `#959595` / `rgba(149,149,149,1)`
  menu text color
 - Mobile  - Subtle elements
  
- **Dark Gray**: `#3c3c3c` / `rgb(60,60,60)`
  - Menu hover background
  - Text on light backgrounds
  
- **Border Gray**: `#282828` / `rgba(40,40,40,1)`
  - Dividers and borders

### Typography
- **Headings Font**: `Julius Sans One` (Google Font)
  - Elegant, classical feel
  - Sizes: 72px (h1), 32px (subheadings)
  
- **Body Font**: `Josefin Sans` (Google Font)
  - Clean, geometric sans-serif
  
- **Fallback**: `Arial, Helvetica, sans-serif`

## Design Characteristics

### Current Layout
- Full-width header with navigation
- Hero section with background image
- Centered text content
- Simple grid for services
- Minimal footer with copyright

### Visual Style
- Classical, elegant, artisan feel
- Focus on craftsmanship and tradition
- Dark header contrasts with light content areas
- Emphasis on the four pillars: Klang, Wert, Erlebnis, Lebensgefühl

### Content Sections
1. **Hero**: Large title with violin background
2. **Navigation**: Horizontal menu with dropdowns
3. **Services Grid**: 5 service cards
4. **Instrument Gallery**: Alemannische & Italienische models
5. **About**: Personal story and craftsmanship
6. **Contact**: Form and contact info

## Recommendations for Redesign

### Modern Aesthetic Directions

1. **Option A - Modern Artisan**
   - Keep warm wood tones and classical elegance
   - Use more whitespace
   - Improve typography hierarchy
   - Add subtle animations

2. **Option B - Minimalist Luxury**
   - Clean, white space dominant
   - Accent with deep red
   - Large, high-quality instrument photos
   - Refined typography

3. **Option C - Dark Mode Elegance**
   - Use dark theme (current header as base)
   - Wood/soundhole accents
   - Dramatic lighting effects
   - Premium, exclusive feel

### Key Improvements to Consider
- Mobile responsiveness (currently basic)
- Image optimization and lazy loading
- Better navigation UX
- Contact form improvements
- SEO optimization
- Page speed optimization
